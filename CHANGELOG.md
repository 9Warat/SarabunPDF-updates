# SarabunPDF — บันทึกการเปลี่ยนแปลง

## 3.0.0
- หมายเลขลำดับ (Step) — คลิกวาง 1,2,3… เลือกรูปทรง (วงกลม/สี่เหลี่ยม/หมุด), สี, ชนิดลำดับ, ขนาด, เงา · ลบตัวกลางแล้วเรียงเลขใหม่อัตโนมัติ
- โหมดนำเสนอ: ซูมเข้า/ออก (Ctrl+ล้อ + ปุ่มซูม) และลากเลื่อนดูตอนซูม
- เส้นโยงกล่องความคิดเห็น (Sticky Note) จากจุดที่คลิกไปยังกล่อง
- แท็บ "แปลงไฟล์" รวมเครื่องมือแปลง/ส่งออก · รวม Ghostscript มากับตัวติดตั้ง (PDF/A + บีบอัดคงข้อความพร้อมใช้)
- รวมฟีเจอร์ตั้งแต่ 2.x: Recent Files, ตั้งค่าโปรแกรม, โทเคนวันที่ พ.ศ., autosave เบื้องหลัง, แยกตามสารบัญ, ขาวดำ, ฟอร์ม/เลเยอร์/เทียบภาพ, ส่งออก Office/PowerPoint, OCR ค้นหาได้ ฯลฯ
